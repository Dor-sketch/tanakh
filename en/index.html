First Time Use

Welcome to the Online Bible! You can select a book, chapter, and read verses in Hebrew.

You can search for a term using the search bar, and navigate between chapters using the "Previous Chapter" and "Next Chapter" buttons.

The system remembers where you last read, and will open at that point next time you enter.

About

The Online Bible was developed non-profit with the aim of making the Bible accessible in a user-friendly way.

The software operates with independently developed algorithms and is based on data from the Sefaria Project.

The original code is available on GitHub.
